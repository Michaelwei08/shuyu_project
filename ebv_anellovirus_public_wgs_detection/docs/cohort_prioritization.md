# Cohort Prioritization

## Ranking Rules

- Data accessibility: open data ranks above controlled-access data; controlled-access WGS with clear accession ranks above publication-only cohorts.
- Disease relevance: HL, DLBCL, and PTLD are all in scope; tumor WGS ranks above non-tumor WGS for EBV tumor contribution.
- Sample count: larger cohorts rank higher if raw data and metadata are available.
- Metadata completeness: EBV annotation, treatment, transplant status, survival, and batch information increase priority.
