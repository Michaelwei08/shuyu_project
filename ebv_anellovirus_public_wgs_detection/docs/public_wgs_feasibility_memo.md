# Public WGS Viral Detection Feasibility Memo

## Objective

Detect EBV, other herpesviruses, and anelloviruses from public WGS in HL, DLBCL, and PTLD to validate and expand the viral subtype paper.

## Current Status

- HL has a high-priority EGA WGS cohort candidate but requires controlled-access approval.
- DLBCL has candidate EBV-positive WGS publications, but data accessions still need confirmation.
- PTLD has directly relevant viral/metagenomic/cfDNA studies, but no immediately runnable public WGS cohort has been confirmed.

## Ready For Inclusion

No public WGS cohort is currently marked `ready_now`. The nearest actionable cohort is the controlled-access HL EGA WGS dataset.

## Recommended Next Step

Request or confirm access to the HL EGA WGS cohort while continuing DLBCL and PTLD accession discovery. In parallel, keep the viral detection workflow ready so the first approved data subset can be run immediately.
