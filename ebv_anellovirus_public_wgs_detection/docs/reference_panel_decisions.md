# Reference Panel Decisions

## Primary Panel

The primary pilot panel should include EBV type 1, EBV type 2, HHV-6A, HHV-6B, HHV-7, CMV, KSHV, HSV-1, and representative TTV/anellovirus references.

## Nature 2026 Panel

Include the Nature 2026 31-virus panel as a comparison panel once its exact accession list is extracted or reconstructed. This will make subtype replication more directly comparable to the published virome GWAS/MR framework.

## Broader RefSeq Panel

Run a broader RefSeq viral sensitivity screen after the primary panel is working. The broader panel should be used for discovery and artifact review, while the primary panel should remain the main source for EBV/anellovirus subtype calls.

## Decision

- Primary analysis: targeted herpesvirus plus anellovirus panel.
- Comparability analysis: Nature 2026 panel.
- Sensitivity analysis: broader RefSeq viral panel.
