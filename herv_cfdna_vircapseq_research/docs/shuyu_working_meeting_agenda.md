# Shuyu Working Meeting Agenda

## Goal

Collect the minimum information needed to benchmark HIV, HTLV, and HERV detection on targeted sequencing and WGS data.

## Questions To Resolve

- Which samples are known HIV positive?
- Which samples are known HTLV positive?
- Which samples are healthy negative controls?
- Which samples are from the HIV cohort, HL cohort, and Takeshi's HTLV cohort?
- Which samples have targeted sequencing?
- Which samples have WGS?
- Which files are ready now: FASTQ, BAM, or CRAM?
- Which reference build was used for existing alignments?
- Which metadata fields can be shared for benchmarking?

## Files To Request

- HIV cohort targeted-sequencing manifest.
- Healthy-control targeted-sequencing manifest.
- Takeshi HTLV cohort targeted-sequencing manifest.
- 60-sample WGS manifest.
- Any prior HIV, HTLV, EBV, or HERV calls.
- Any existing sample-level QC table.

## Immediate Output From Meeting

- A populated copy of `data/manifests/sample_manifest_template.csv`.
- A decision on the first benchmark subset.
- A list of files that can be analyzed first.
- A list of access or transfer blockers.
