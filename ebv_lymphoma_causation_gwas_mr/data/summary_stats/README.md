# Summary Statistics

Place external GWAS summary statistic manifests here. Do not store restricted summary statistics unless their data-use terms allow local storage in this workspace.

Recommended manifest columns:

- `trait`
- `source`
- `accession_or_url`
- `case_count`
- `control_count`
- `ancestry`
- `genome_build`
- `access_status`
- `local_path`
- `notes`
